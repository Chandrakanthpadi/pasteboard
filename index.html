How to use Pasteboard

Arrange screenshots into a board, annotate them, and export to PDF or Word — all in your browser.

1 · Add content

  • Paste a screenshot — capture with Snipping Tool / PrtScn, then press Ctrl+V anywhere on the page.
  • Paste files — copy PDFs, Word, Excel, PowerPoint, or Outlook .msg files in File Explorer and paste; each becomes an attachment card. Multiple files at once keep their order.
  • Text cards — paste plain text, click Add a text card, or the + button between cards. URLs you type become clickable links in the exports.

2 · Organize with sections & steps

  • Sections → Steps → Cards. A Section is a top-level box; inside it you add Steps, and each step holds the cards. Use Add Section (Alt+S), each section's + Step button (Alt+T), and + Card on any section/step head.
  • Numbering is automatic: sections 1, steps 1.1, cards 1.1.1. Cards placed loosely (at the top of a section or above all sections) stay unnumbered and always sit on top.
  • Collapse / expand by clicking a section or step bar. A section's double-chevron toggles all its steps at once; the navbar button collapses or expands everything.
  • Pin (the 📌 on any section or step head) sets where pastes land — into that step, or as a loose card in that section. With nothing pinned, pastes flow into the last section's last step.
  • Reorder by dragging a card into any step/section (or the loose area); drag a section's or step's handle to move the whole box with its contents.
  • Multi-select with each card's checkbox, then Group as Section, Group as Step, or Delete the batch (one undo reverses it).
  • Two deletes on each divider: ✕ removes just the divider and keeps its cards; the 🗑 trash deletes the section/step and all its cards.

3 · Edit & annotate

  • Click any card to open it. Use ← / → to move between cards (across steps and sections too).
  • Add a caption under an image, or edit text-card content.
  • On images, use the toolbar tools — Pencil, Highlighter, Rectangle, Text, Eraser — with per-tool color. Annotation ↶ Undo / ↷ Redo is independent of the main undo.

4 · Export & save

  • Export PDF and Export Word both — review the preview, set a file name & author, and get a cover page plus a Contents page (sections & steps with page numbers). Everything — the document, attachments, and the optional editable .json — comes down as a single .zip.
  • In Word, if the Contents page looks empty, right-click it and choose Update Field to fill in the page numbers.
  • Save / Open — store your work as a .json project and reload it later to keep editing.
  • Download app — save this whole app as one HTML file to use offline.

Keyboard shortcuts

  • Ctrl+V Paste
  • Ctrl+Z Undo
  • Ctrl+Y Redo
  • Alt+S Add section
  • Alt+T Add step
  • ← / → Previous / next card
  • Esc Close a dialog
