How to use Pasteboard

Arrange screenshots into a board, annotate them, and export to PDF or Word — all in your browser.

1 · Add content

  • Paste a screenshot — capture with Snipping Tool / PrtScn, then press Ctrl+V anywhere on the page.
  • Paste files — copy PDFs, Word, Excel, PowerPoint, or Outlook .msg files in File Explorer and paste; each becomes an attachment card. Multiple files at once keep their order.
  • Text cards — paste plain text, click Add a text card, or the + button between cards. URLs you type become clickable links in the exports.

2 · Organize with sections

  • Add Section creates a divider; every card after it belongs to that section until the next divider.
  • Click a section bar to collapse / expand it; the navbar button collapses or expands all at once.
  • Pin a section (the 📌 button on its header) to make pastes land at the end of that section instead of the page bottom.
  • Reorder by dragging any card; drag a section's handle to move the whole section with its cards.
  • Multi-select with each card's checkbox, then Group as Section or Delete the whole batch (one undo reverses it).

3 · Edit & annotate

  • Click any card to open it. Use ← / → to move between cards (across sections too).
  • Add a caption under an image, or edit text-card content.
  • On images, use the toolbar tools — Pencil, Highlighter, Rectangle, Text, Eraser — with per-tool color. Annotation ↶ Undo / ↷ Redo is independent of the main undo.

4 · Export & save

  • Export PDF — review the preview, then set a file name & author for the cover page (with created date). Optionally attach the editable project .json inside the PDF.
  • Export Word — builds a .docx; inserted files travel alongside in a .zip bundle.
  • Save / Open — store your work as a .json project and reload it later to keep editing.
  • Download app — save this whole app as one HTML file to use offline.

Keyboard shortcuts

  • Ctrl+V Paste
  • Ctrl+Z Undo
  • Ctrl+Y Redo
  • ← / → Previous / next card
  • Esc Close a dialog
